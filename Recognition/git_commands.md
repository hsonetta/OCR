# BASICS

```
git init
git add <filename>

# add "all files" to staging area
git add .

# diff btw working (saved) and staging (Show changes between commits and working tree)
git diff

# show the working tree status
git status
git commit -m “<message to be sent>”
git log

# take a peek at the latest updates on remote
git fetch

# pull from online to local. Fast forward local copy. (= Fetch + Merge)
git pull
```
